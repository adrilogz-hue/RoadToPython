# RoadToPython

Repositorio de aprendizaje personal para preparar una transición profesional hacia puestos junior relacionados con Python, SQL, Data/IA, QA Automation y Backend.

## Objetivo

Aprender programación desde una base inicial hasta poder desarrollar proyectos sencillos, usar Git/GitHub y preparar un portfolio técnico en un plazo de 4 a 6 meses.

## Contenido actual

### 01_python_basico

Ejercicios básicos de Python:

- print()
- variables
- f-strings
- input()
- int()
- condicionales if / elif / else
- operadores and / or / not
- listas
- diccionarios
- bucles for
- mini programa final de perfil de estudio

## Tecnologías usadas

- Python
- Git
- GitHub
- Visual Studio Code
